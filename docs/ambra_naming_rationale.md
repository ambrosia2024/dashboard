# Why We Named the Assistant "Ambra"

We chose **Ambra** as the voice of Ambrosia because it shares the same root while feeling more human and conversational.

The name **Ambrosia** comes from Greek tradition, where it is associated with the idea of “immortal” or “divine nourishment” - something precious, trusted, and sustaining.

**Ambra** keeps that heritage, but sounds lighter and more personal: clear, warm, and memorable.

There is also a subtle European meaning layer: in several languages, *ambra* evokes **amber** - a material linked with clarity, preservation, and time.

That fits this assistant’s role: helping users interpret chart data over time, preserve context, and turn complex information into practical insight.

So **Ambra** is not only a shortened project name. It is the project’s human interface - rooted in heritage, designed for trust, and built for clarity.
